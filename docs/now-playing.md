# Now Playing

using [Tuna](https://obsproject.com/forum/resources/tuna.843/)

## Customizing

here are some stuff you can customize via the CSS tab in OBS:

### Font / Text

```css
figure p {
	font-family: 'Comic Sans MS' !important;
	text-transform: lowercase;
}
```

## Remove Wave Form

```css
#waveform {
	visibility: collapse;
}
```
