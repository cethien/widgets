## Customizing

you can customize using the css-field in OBS Browser Source like this#️⃣

```css
body {
	background-color: transparent;
	margin: 0px auto;
	overflow: hidden;
	font-family: 'Comic Sans MS';
	background-image: linear-gradient(
		to right,
		rgba(0, 0, 0, 0),
		rgba(77, 125, 100, 0.5)
	);
}
```
